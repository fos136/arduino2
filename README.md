도둑 방지 경보 시스템 계획서 (간단 버전)
1. 프로젝트 개요

이 프로젝트는 아두이노와 적외선 센서를 이용해 문이 열릴 때 침입을 감지하고 경보를 울리는 시스템을 만드는 것이다. 또한 Processing과 앱 인벤터를 활용해 집 밖에서도 침입 알림을 받을 수 있는 기능을 추가한다.

2. 동작 원리

IR 송신 다이오드가 문에 적외선을 계속 쏜다.

적외선이 문에 반사되어 IR 수신기로 들어온다.

문이 열리면 적외선이 반사되지 않아 수신 신호가 사라진다.

아두이노가 이를 침입으로 판단하여 LED 점등 + 피에조 스피커 경보음을 울린다.

동시에 Processing → 앱 인벤터로 침입 메시지를 보내 집주인 스마트폰에 알림을 전달한다.

3. 사용 부품

Arduino Uno

IR 송신 다이오드, IR 수신기

1kΩ 저항

LED

피에조 스피커

Processing / App Inventor

4. 주요 기능

침입 감지: IR 반사 신호 변화로 문 열림 즉시 판단

현장 경보: LED와 스피커로 큰 경고음 출력

원격 알림: 스마트폰으로 침입 이벤트 즉시 전달

확장 가능: 자동 신고 기능 등 추가 가능

5. 기대 효과

문이 열리는 즉시 빠른 감지 가능

집을 비워도 스마트폰으로 즉각 알림

간단한 회로로 저비용 보안 시스템 구현
